# 💼 Oasis Infobyte Internship Projects (OIBSIP)

> 🚀 A collection of Machine Learning & Web Applications developed during the Oasis Infobyte Data Science Internship.

---

## 📌 About the Internship

This repository contains all the tasks completed as part of the **Oasis Infobyte Data Science Internship Program (OIBSIP)**.

Each project demonstrates real-world applications of **Machine Learning, Data Science, and Web Development using Flask**.

---

## 🧠 Projects Overview

### 🌸 Task 1: Iris Flower Classification

A Machine Learning web app that predicts the species of an Iris flower.

✔ Logistic Regression Model
✔ Real-time prediction using Flask
✔ Clean UI with input-based prediction

📂 Folder: `Task1_Iris_Classification`

---

### 🚗 Task 2: Car Price Predictor

A web application that predicts car prices and compares vehicles.

✔ Random Forest Regressor
✔ Price prediction + comparison feature
✔ Dynamic UI with fuel-based images

📂 Folder: `Task2_Car_Price_Predictor`

---

### 📩 Task 3: Spam Message Classifier

An NLP-based application that classifies messages as Spam or Not Spam.

✔ Naive Bayes Classifier
✔ TF-IDF Vectorization
✔ Real-time text prediction

📂 Folder: `Task3_Spam_predictor`

---

## 🚀 How to Run Projects

### 1️⃣ Clone Repository

```bash
git clone https://github.com/sumit312-cpu/OIBSIP.git
```

---

### 2️⃣ Navigate to any task folder

```bash
cd OIBSIP/Task1_Iris_Classification
```

---

### 3️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

---

### 4️⃣ Run the application

```bash
python app.py
```

---

### 5️⃣ Open in browser

```
http://127.0.0.1:5000/
```
